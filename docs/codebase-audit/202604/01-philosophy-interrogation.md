# Phase 1: Philosophy Interrogation

> Status: not started

## Method

For each of the 12 principles in PHILOSOPHY.md, answer three questions:
1. **Edge case pressure test** — what scenario makes this principle hurt?
2. **Consequences audit** — are there unstated consequences? Are listed consequences actually implemented?
3. **Principle conflicts** — where does this principle pull against another?

Severity ratings: `cosmetic` (documentation issue only), `design-tension` (acceptable trade-off but should be documented), `actual-violation` (code or spec contradicts the principle).
